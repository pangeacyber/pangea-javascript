## Pangea OAuth-JS

A vanilla JavaScript client for interacting with Pangea OAuth in the browser.

## Getting started

### Installation

Using [npm](https://npmjs.org/)

```bash
npm install @pangeacyber/oauth-js
```

Using [yarn](https://yarnpkg.com/)

```bash
yarn add @pangeacyber/oauth-js
```
