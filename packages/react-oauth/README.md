## Pangea React OAuth

A React Auth Provider for integrating with Pangea OAuth

## Getting started

### Installation

Using [npm](https://npmjs.org/)

```bash
npm install @pangeacyber/react-oauth
```

Using [yarn](https://yarnpkg.com/)

```bash
yarn add @pangeacyber/react-oauth
```
